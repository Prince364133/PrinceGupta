# Admin Panel Access Documentation

## Admin Panel URL

The admin panel is accessible at:

**https://princekumar.web.app/admin/dashboard**

> **Note**: The login page `/admin/login` will automatically redirect you to the dashboard.

## Access

Access to the admin panel is now **public and unrestricted**. No credentials are required to view or manage the content.

## Admin Features

### Application Management

- `/admin/dashboard` - Main admin dashboard
- `/admin/profile` - Profile management
- `/admin/projects` - Project management
- `/admin/skills` - Skills management
- `/admin/experience` - Experience management
- `/admin/education` - Education management
- `/admin/startups` - Startup management
- `/admin/blogs` - Blog post management
- `/admin/testimonials` - Testimonials management
- `/admin/forms` - Contact form submissions
- `/admin/resume` - Resume management

### Security Note

Since authentication has been removed, anyone with the URL can access and modify the portfolio content. Please ensure you are comfortable with this level of access.
